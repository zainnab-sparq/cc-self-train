Security reference index — cc-self-train curriculum
====================================================

Last synced: 2026-04-19

This file maps common AI-assisted-development security concerns to
the curriculum sections that address them. Scannable when you hit a
concrete threat and need to find "where do I look this up?"

If you want the narrative treatment of these topics, read
docs/SAFETY-AND-TRUST.md (appendix). This file is the lookup index.

If you want an automated audit of your current project's Claude Code
configuration against known footguns, run the /security-review skill.

When in doubt
-------------

The cross-cutting principle is in SAFETY-AND-TRUST.md: slow down. A
permission prompt you read is a dozen times safer than one you
approved.
